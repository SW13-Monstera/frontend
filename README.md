## 👋 소개

AI 기반 서술형 채점 기법을 통해
다양한 유형의 Computer Science 문제를 풀고
스스로 CS 지식을 학습할 수 있는 사이트입니다.

<br/>

## ✨ 기능

- ⭐ AI 기반 서술형 채점
- 📝 단답형, 객관식 문제 풀이
- 🙋‍♂️ 오늘의 문제 추천
- 📊 카테고리별 강약점 분석
